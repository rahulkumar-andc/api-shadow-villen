# Security Policy

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in **Shadow-API Mapper**, please do **not** open a public issue.

### 1. Report Privately
Email us describing the vulnerability with steps to reproduce:

### 2. Response Timeline
- We will acknowledge your report within **48 hours**.
- We aim to provide a fix or mitigation within **14 days**.

### 3. Disclosure
We follow a 90-day responsible disclosure policy. We kindly ask you not to publish details until a patch has been released.

## Scope
This policy covers the Shadow-API Mapper tool itself. Vulnerabilities found in *targets* scanned by this tool should be reported to the target's respective security team/program.
